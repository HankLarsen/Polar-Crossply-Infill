# Ring Test Plan — Polar Cross-Ply vs. Baseline Layups

Objective: measure **hoop strength as a function of layup** using a conventional,
standardized *method*, so the numbers are comparable and defensible.

---

## 1. Standards position — state this accurately

Be precise about what is and isn't standard here. Overclaiming is the fastest way
to lose a technical audience.

- **ASTM D2290** — *Apparent Hoop Tensile Strength of Plastic or Reinforced Plastic
  Pipe by Split Disk Method*, first issued 1964. Current procedures are
  pipe-oriented: **A** = reinforced-thermosetting-resin pipe, **B** = thermoplastic
  pipe of any size, **C** = thermoplastic pipe ≥4.5 in (110 mm) nominal, specimen
  width 44–50 mm.
- **The classic NOL ring is no longer in the standard.** The original Procedure A
  specimen (hoop-wound ring, 5.750 in ID × 0.250 in wide × 0.060 in thick) was
  **deleted in 2000**, when the standard moved from Committee D-30 (Composites) to
  F-17 (Plastic Piping). The fixture is still commercially available and still used
  for filament-wound composite rings, but it is **not** an ASTM standard specimen.
- **Therefore the correct claim for this work is:**
  > "ASTM D2290 split-disk **method**, applied to a **non-standard printed ring
  > specimen**; results reported as **relative comparisons between layups**, not as
  > standard-compliant material values."

Do not write "tested per ASTM D2290" or "NOL ring per ASTM." Both are false.

### Why "apparent" — and why it doesn't invalidate the comparison
As the fixture halves separate, the ring's contour changes across the split and a
**local bending moment** is imposed. The standard's formula counts only the tension
component, so reported *apparent* strength understates true tensile strength —
published work puts the gap at **up to 50% or more**.

This matters for absolute numbers and **not** for this program, because every layup
uses **identical geometry, fixture, and procedure**, so the bending bias is
**common-mode and cancels in layup-to-layup ratios**. Report ratios. Treat absolute
apparent strength as indicative only, and say so.

## 2. Specimen

Generated by `make_specimens.py`. Defaults (all configurable):

| Parameter | Default | Notes |
|---|---|---|
| Outer diameter | 80 mm | fits a 220×220 bed with room for a batch |
| Inner diameter | 50 mm | |
| Wall thickness | 15 mm | thick enough for the radial plies to be expressible |
| Axial width (print height) | 10 mm | ring axis = print Z |
| Wall area per side, A | 150 mm² | used in σ = P / 2A |

**Wall thickness is a deliberate trade.** A thin wall keeps D2290's thin-wall
assumption honest but makes radial plies degenerate (spokes only a few mm long). A
thick wall makes the pattern expressible but increases bending error. 15 mm favors
*measuring the pattern*; since bending bias is common-mode, ratios survive it.

**Orientation:** print with the ring axis vertical (flat on the bed). Layers then
stack along the ring's axial width, and each layer is an annulus carrying hoop or
radial roads — exactly the layup under test.

**No solid caps.** `solid_cap_layers = 0` in the specimen config on purpose: caps
would add hoop-oriented skins to *every* specimen and mask the difference being
measured. Walls are minimal and identical across layups (common-mode).

### Reduced sections — required before testing
D2290 specimens carry drilled half-circle **reduced sections** so failure occurs at
a defined location away from the split, where bending is worst. The generator does
**not** model these. Without them, failure location is uncontrolled and scatter
rises. Either model two diametrically-opposed reduced sections into the specimen,
or machine them post-print — and apply them **identically to every layup**.

## 3. Test matrix

| Layup | `ply_sequence` | Role |
|---|---|---|
| `crossply` | `R,H` | the pattern under test |
| `hoop` | `H` | hoop-optimal reference (upper bound for this test) |
| `radial` | `R` | radial-optimal reference (expected worst in hoop) |
| `grid` | `G` | ±45° rectilinear — **the stock slicer baseline** |
| `solid` | `H` dense | upper-bound reference |

- **Replicates:** ≥5 per layup (25 specimens at default settings).
- **Mass matched:** the generator tunes infill density so all layups use within
  ~1% of the same filament. Equal mass is what makes the comparison fair.
- **Controls:** same machine, same material lot, same nozzle/temperature/cooling,
  printed in as few sessions as possible. Record everything.

```bash
python make_specimens.py --replicates 5
# then set temps + your printer's start/end g-code before printing
```

## 4. Fixture and procedure

- **Fixture:** split-disk — two half-disks (D-shaped) that seat inside the ring and
  are pulled apart in a load frame. Must be stiff steel/aluminium; a printed fixture
  will deform and corrupt the result. Commercial D2290 fixtures exist; a shop-made
  equivalent is acceptable **if fully documented**.
- **Load frame:** universal testing machine preferred. If improvising, you still
  need a **calibrated load cell** and a **controlled, constant crosshead rate** —
  without both, the numbers are not comparable to anything, including each other.
- **Rate:** fix one rate and use it for every specimen; report it.
- **Conditioning:** same temperature/humidity for all specimens; report it.

### Calculation
Apparent hoop tensile strength:

```
sigma_apparent = P_max / (2 * A)
```

where `P_max` is peak load and `A` is the cross-sectional area of **one** wall
section (wall thickness × axial width; 150 mm² at defaults). The factor 2 accounts
for the ring carrying load in two places.

## 5. What this test does and does not show

**Does show:** relative hoop-direction strength across layups. Expected ordering is
`hoop ≥ crossply > grid > radial`. The key number is the **hoop penalty of
cross-ply** — how much hoop strength you trade for gaining radial capability — and
whether cross-ply beats the **stock grid baseline**.

**Does not show:** the *benefit* of the radial plies. A ring test loads hoop only.
Radial capability must be measured separately.

### Complementary tests
| Test | Standard | Measures |
|---|---|---|
| Short-beam shear | **ASTM D2344** | interlaminar shear (the documented weak mode) |
| Tensile coupons 0°/90°/±45°/Z | **ASTM D638** | orthotropic constants → feeds FEA |
| Bore press-fit / pin push-out | custom — document fully | bore hoop capacity at the real failure site |
| Torque hub→rim | custom — document fully | in-plane shear; the known worst case |

No standard covers hub-to-rim torsion on a printed disc, so that fixture and method
must be described in full rather than cited.

---

## 6. Reporting

Report, in this order: method statement (with the §1 caveats stated plainly) →
specimen geometry and process parameters → per-layup mean ± SD with n → **ratios
between layups** → failure locations and modes → limitations.

Publish the raw load data alongside the summary. State machine, material, and every
process parameter. If failure occurred at the split rather than a reduced section,
say so — it changes how the number should be read.
